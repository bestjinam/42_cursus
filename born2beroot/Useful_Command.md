- systemctl 
	- systemd(system daemon)을 관리하는 명령어
	- systemd 는 unix 시스템이 부팅후 가장 먼저 생성되어 다른 프로세스를 실행하는 init역할을 대체 하는 데몬
- systemctl status 서비스명 : 상태 확인
- systemctl enable 서비스명 : 부팅시 자동 시작
- systemctl start
- systemctl reboot
- systemctl poweroff
